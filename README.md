# 智能OCR系统

一个支持实时摄像头识别和图片文件识别的智能OCR系统，采用模块化设计，具有自动降级机制。

## 功能特点

1. **双模式工作**
   - 图片模式：处理单张或多张图片文件
   - 实时模式：通过摄像头实时识别文本

2. **自动降级策略**
   - 主检测器：YOLO
   - 备用检测器：传统图像处理算法
   - 当YOLO检测效果不佳时自动切换到传统方法

3. **多语言支持**
   - 支持中文简体（ch_sim）
   - 支持英文（en）


4. **多种使用方式**
   - 图形界面（GUI）：用户友好的可视化操作
   - 命令行接口（CLI）：适合自动化处理
   - Python API：可直接在代码中调用
